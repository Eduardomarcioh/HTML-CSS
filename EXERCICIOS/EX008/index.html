Exemplo com código Phyton

Esse exemplo foi inserido a tag <pre> e depois um código phyton usando a tag <code>

num = inte(input('Digite um numero')) if num % 2 == 0: print(f'0 número{num} é PAR) else: print(f'0 número {num} é ÍMPAR) print('Fim do programa')

Vemos que a tag ficou em formato de texto seguido, não um embaixo do outro. Para isso, vamos envolopar usando o EM (CTRL+SHIFT+P) em PRE. Desse jeito:

        
            num = inte(input('Digite um numero'))
            if num % 2 == 0:
                print(f'0 número{num} é PAR)
            else:
                print(f'0 número {num} é ÍMPAR)
            print('Fim do programa')

Dica:

Se quiser tirar esse espaçamento na tag PRE, você seleciona o texto (sem as tags) segura a tecla SHIF+TAB, ele volta o espaçamento ou apenasa o TAB que ele acrescenta. EXEMPLO:

        
    num = inte(input('Digite um numero'))
    if num % 2 == 0
        print(f'0 número{num} é PAR)
    else:
        print(f'0 número {num} é ÍMPAR)
    print('Fim do programa')

Dentro da tag PRE você também pode criar espaçamento, usando o ENTER ao inves de digitar a tag BR. Não precisa fazer isso no HTML, pois isso você consegue fazer no CSS. Mas é possível!
