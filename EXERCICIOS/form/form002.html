Sexo



Nesse formulário foi utilizado um novo atributo chamado REQUIRED. Ele é utilizado quando o usuário precisa informar aquele dado, torando obrigatório.

Também foi usado uma atribuição para o tamanho minino e maximo de um dado. Nesse caso, foi solicitado usando MINLENGTH "5" e MAXLENGTH"25" . Isso significa que os dados tem que ter no minimo 5 letras e no máximo 25.

Usando o PLACEHOLDER você coloca a mensagem dentro da caixa.

Usando a tag FIELDSETvocê consegue colocar o formulário dentro de uma box, onde você pode customizar usando CSS.

No formulário de SEXO o nome do input tem que ser o mesmo, pois você so poderá escolher uma das duas opções. Caso o nome fosse diferente, tanto o masculino quanto o femino seriam marcados.
