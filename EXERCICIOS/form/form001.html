Meu primeiro Formulário

Nesse código podemos observar que no FORM foi adicionado o ACTION como (Cadastro.php) pelo fato que esse dados terãp que ser enviados para um outro servidor, no caso, utilizado o php.

Também foi adicionado a tag chamada (autocomplete"off"). Essa tag é utilizada para que as informações dentro campo de preenchimento não fiquem salvas.

A TAG LABEL, foram utilizadas envelopando tanto o SOBRENOME e o NOME. Dentro do LABEL você coloca o ID do input. Isso irá facilitar o algoritimo de busca

Na TAG METHOD="" você terá duas opções: GET e POST. Quando utilizar as duas? O GET é o mais padrão e você irá utilizar quando os dados a serem enviados, não sejam sensiveis. O POST é o contrário. O POST é usado em envios de senhas, imagens e envio de arquivos por exemplo. Para que os dados sensíveis fiquem mais seguros, deve ser utilizado o PHPS, pois ele criptográfa esse tipo de informações em dados mais sensíveis.
