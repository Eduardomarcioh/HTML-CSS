Usando links externos

Link externo é um link que vai ter levar para outros sites

Por exemplo: Canal do alanzoka

O link do canal do alanzoka foi envolepado usando a CRTL+SHIFT+P +a. A Tag "a" ela é chamada de ancora, pois nela onde fica o link de direcionamento para o site final desejado.

Na area de HREF foi colocado a URL do site do Alanzoka.

Colando Link para ser aberto em uma outra aba

Veja que, ao clicar no link da live do alanzoka, ela subescreveu o site que ja estava aberto, ou seja, você perde acesso ao site anterior

Por isso, devemos acrescer junto a tag "a" as seguintes informaçoes "target="_blank" e "rel="external"

Usando links internos

Links internos são aqueles que fazem parte do site, que vão levar a áreas do seu próprio site.

Criamos uma nova pagina dentro do mesmo HTML chamada de "pag002.html"

Está é primeira página do site. Para acessar a segunda página, clique em segunda página

Dica: Usando o studio code, você pode usar uma tecla de atalho CRTL+SPACE que ele te mostrará as páginas.

Quando eu faço um link interno, eu não preciso indicar a URL inteira.

Podemos colocar também dentro da tag "a" o "rel="next" na primeira página e "rel="prev" na segunda página. Visualmente não muda nada, porém adiciona uma semântica ao site. Por exemplo: Clique aqui
