Imagem dinâmica

Tente abrir esse site em vários dispositivos diferentes ou simplesmente aumente e diminua o tamanho do seu navegador.

Foi usado a tag SOURCE MEDIA TYPE e mudei o width para "max". No SRCSET foi adicionado a imagem média e no TYPE adicionamos o final PNG.

Isso significa que, quando a tela for maior que 1050, a imagem GRANDE irá aparecer. Tela até 1050 irá aparacer a imagem MÉDIA e Imagem menor em tela máxima em 750px

ATENÇÃO!!

Os SOURCES devem ser adicionado sempre acima da IMG uploadiada primeira. Por exemplo, a IMG GRANDE foi adicionado usando a tag IMG, porém a imagem média e pequena, foram adicionadas usando a tag SOURCE!!!

As imagens foram editadas usando GIMP e as dimensões das imagens foram P:80px M:700px G:1000px

Reproduzindo Áudio

Vamos aprender a reproduzir áudios em HTML5 usando a tag AUDIO, selecionar o áudio usando CTRL+SPACE e adicionar junto a tag o CONTROLS AUTOPLAY.
